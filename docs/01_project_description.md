# 📊 Jordan Logistics Performance Analysis (LPI)

## 📌 Project Overview
This project analyzes the Logistics Performance Index (LPI) with a focus on Jordan.  
The goal is to understand the country's logistics performance, identify weaknesses, and simulate improvement scenarios.

---

## 🎯 Objectives

- Analyze LPI trends over time  
- Compare Jordan with other countries using clustering  
- Identify the weakest logistics indicators  
- Forecast future LPI scores  
- Build a What-if simulation tool  

---

## 🏢 Business Problem

Jordan has shown relatively slow improvement in logistics performance compared to peer countries.  
This affects trade efficiency, supply chains, and economic competitiveness.

---

## ❓ Key Questions

- Why did Jordan drop from Mid-High to Mid-Low cluster?  
- What are the weakest logistics indicators?  
- How can improving these indicators impact overall performance?  

---

## 💡 Project Contribution

This project transforms raw data into:
- Insights about Jordan’s logistics performance  
- A forecasting model  
- An interactive decision-support tool (Streamlit app)  

---

## 📈 Expected Outcome

- Identify key weaknesses (Customs & International Shipments)  
- Provide improvement scenarios  
- Support decision-making using data-driven insights
